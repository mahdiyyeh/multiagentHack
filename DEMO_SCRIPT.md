# SpaceRaid — Demo Script

**Length:** 3 minutes  
**Sponsors to highlight:** Tavily, Senso, Gensyn, Cursor

---

## Before you go on stage

- [ ] Backend: `./run_backend.sh`
- [ ] Frontend: `cd frontend && npm run dev`
- [ ] Room photo ready
- [ ] Start on floor plan → **Common room**

## One-liner

> SpaceRaid audits your space from a photo, raids the live web with Tavily, verifies picks with Senso, receipts the shortlist with Gensyn — and was built with Cursor.
